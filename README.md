# 友利奈緒575bot

## 概要

[@kokodeikku_bot](https://twitter.com/kokodeikku_bot/) がつぶやいたツイートの末尾を「友利奈緒」に変換してつぶやくbot []@tomori_nao_575](https://twitter.com/tomori_nao_575) のソースです。

## バージョン

Ver 1.0

## 動作環境

Python、tweepyモジュールのインストールが必要(説明略)

## 動作確認環境

* OS: Ubuntu 14.04.3 LTS
* Python: 2.7.6
